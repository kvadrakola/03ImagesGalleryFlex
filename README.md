# Responsive Image Gallery

A simple responsive image gallery built with **HTML** and **CSS Flexbox**.

## About

A webpage showcasing magical and bioluminescent images. Gallery cards adapt to screen size:

- **Mobile** — 1 image per row
- **Tablet (≥768px)** — 2 images per row
- **Desktop (≥1024px)** — 4 images per row

## Usage

1. Download or clone the repository.
2. Open `index.html` in any modern browser.

No servers or installations required — it works right away.
